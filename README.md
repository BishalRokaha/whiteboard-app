## âš™ï¸ How It Works

- Mouse events such as `mousedown`, `mousemove`, and `mouseup` are used to track user interaction.
- When the mouse is pressed, drawing begins by capturing cursor coordinates.
- As the mouse moves, continuous lines are drawn between points using the Canvas API.
- React hooks manage drawing state and control logic, while the canvas handles rendering.
